How to read this. A “block” is a provider’s bookable availability window. Blocks are generic — one block usually allows several consultation types, online and offline. Booked slots = exact count of realised appointments (completed + no-show); offered slots = offered time ÷ that provider’s average appointment length (approx); blank/wasted = offered − booked. Utilisation % is exact (booked ÷ offered time). In the table, capacity columns ignore Type/Location; outcome columns respect them. Only providers currently accepting new patients are included.
